# Where can I donate to ?

This repository is a compilation of various organizations and causes that accept donations or contributions. It includes links to their websites/repos, as well as information about the work they do and how donations and contributions can help.

Open Source Software is the cornerstone of modern software development, we build on the shoulders of the ones who came before us, and paved the way for us to build incredible things that makes the world go around.

We, as developers, have the social responsibility to give back to the ones who made our work possible, and by supporting them we are continuing the cycle of innovation and progress that has made our industry what it is today, by allowing them to continue to improve, secure and maintain the software that powers our digital lives.

Together we are much stronger, and can go much further than were we've gotten so far, we can do this humans!

## How to contribute?

open an issue, or a pull request, to add more projects, or to update the information of the existing ones, and if you have any questions or suggestions please feel free to reach out.

#### OpenSSL

OpenSSL is what allows us to have TLS and SSL protocols, not only that but it also allows us, to generate private keys, create CSRs, install SSL Certificates and manage PKI.

- https://github.com/openssl

#### NodeJS

A versatile JavaScript runtime that allows us to code web servers, real-time apps, CLI tools, database interaction, file management, and much more probably.

- https://github.com/nodejs

#### OWASP

Organization dedicated to improving secure software practices.

- https://github.com/OWASP

#### Blender

3D pipeline software, used for 3D modeling, animation, rendering, and more.

- https://www.blender.org/

#### VideoLan Org

Creators of VLC media player, and other free and open-source cross-platform multimedia players.

- https://www.videolan.org/

#### Linux Foundation

An organization that supports the growth of open-source software and collaborative development.

- https://www.linuxfoundation.org/about/donate

#### Apache Foundation

Organization that supports projects, including the Apache HTTP Server, Apache Hadoop, and many others.

- https://www.apache.org/foundation/individual-supporters

#### F-Droid

Free repository of open-source Android applications.

- https://f-droid.org/en/

#### KDE

A community that develops free and open-source software, including the KDE Plasma desktop environment.

- https://kde.org/

#### Mozilla Foundation

An organization that promotes an open and accessible internet, and develops the Firefox web browser.

- https://www.mozillafoundation.org/

#### Signal

A secure messaging app developed by the Signal Foundation.

- https://signal.org/

#### Nextcloud

- https://nextcloud.com/

#### OBS Studio
